# ClickUp Card Description

## Title
Calculator Web Application - Full Stack React + Node.js

## Description
Build a responsive full-stack calculator web application using React for the frontend and Node.js for the backend.

### Core Requirements
- Standard arithmetic operations
- Scientific functions: percentage, square root, exponents, sign toggle
- Trigonometric functions with degree/radian mode
- Logarithmic functions: log and ln
- Constants: pi and e
- Memory operations: M+, M-, MR, MC
- History panel showing last 10 calculations
- Keyboard support for numbers, operators, enter, and backspace
- Friendly error handling for invalid expressions and math edge cases

### Technical Expectations
- Responsive UI for mobile, tablet, and desktop
- Clean modular React components
- Structured Node.js backend using routes/controllers/services
- Minimal external libraries
- Safe evaluation logic (no unsafe execution)

### Deliverables
- React frontend
- Node.js backend API
- Documentation covering architecture, API, DB schema, and tasks
- Ready-to-run project structure

### Acceptance Criteria
- Users can perform both standard and scientific calculations reliably
- App remains stable for invalid input and edge cases
- Last 10 calculations are visible in history
- Keyboard and button interaction both work smoothly
- UI is responsive across mobile, tablet, and desktop
