# ClickUp Card Description

## Title
Calculator Web Application - React + Node.js Scientific Calculator

## Strong Description
Build a fully functional **responsive calculator web application** using **React 18** for the frontend and **Node.js 24.13.1** for the backend.

The product must support:
- Basic arithmetic: `+`, `-`, `*`, `/`
- Advanced operations: `%`, `sqrt`, `^`, sign toggle
- Trigonometric functions: `sin`, `cos`, `tan`
- Angle mode toggle: `Degrees / Radians`
- Logarithmic functions: `log`, `ln`
- Constants: `pi`, `e`
- Memory functions: `M+`, `M-`, `MR`, `MC`
- Calculation history panel showing the latest 10 entries
- Keyboard support for numbers, operators, Enter, and Backspace
- Friendly error handling for invalid expressions, divide-by-zero, and invalid scientific domains

## Delivery Expectations
- Clean modular React component structure
- Structured Node.js backend with controllers/services
- Responsive UI for **mobile, tablet, and desktop**
- Minimal external dependencies only
- Ready-to-run frontend + backend project setup
- Safe calculation handling with validation

## Acceptance Criteria
- Users can perform both standard and scientific calculations
- UI updates instantly and feels responsive
- Keyboard and button interactions both work reliably
- Angle mode visibly toggles and affects trig calculations
- Memory controls function correctly
- History retains only the latest 10 entries
- Invalid inputs never crash the application

## Recommended Subtasks
1. Project setup
2. Responsive calculator UI
3. Keyboard input handling
4. Memory + history features
5. Backend calculate API
6. Error handling and validation
7. Integration testing and responsive QA
