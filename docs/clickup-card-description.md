# ClickUp Card Description

Build a responsive calculator web application using React (frontend) and Node.js (backend). The app must support standard arithmetic, scientific functions, trigonometric and logarithmic operations, constants, memory controls, keyboard interaction, and a rolling history panel for the last 10 calculations.

## Scope
- Basic operators: +, -, *, /
- Advanced operations: %, sqrt, exponent, sign toggle
- Scientific functions: sin, cos, tan, log, ln
- Constants: pi, e
- Degree/Radian toggle
- Memory: M+, M-, MR, MC
- History panel with expression + result
- Strong error handling for invalid/domain cases
- Mobile-first responsive UI

## Technical Expectations
- React 18 frontend
- Node.js 24.13.1 backend
- Minimal dependencies only
- Safe expression evaluation (no unsafe eval)
- Modular frontend and backend structure

## Done When
- Calculations are fast and stable
- Keyboard and button input are both supported
- History is capped at 10 items
- UI works cleanly on mobile, tablet, and desktop
- Invalid input does not crash the app
